# Button

> Basic button component

## Usage

```jsx
  import { Button } from '@omnious/kit';

  <Button>
    <span>This is `<Button />`!</span>
  </Button>
```

## Props

| Name     | Type   | Required | Default   |
|----------|--------|----------|-----------|
| disabled | bool   | no       | -         |
| dark     | bool   | no       | -         |
| size     | string | no       | 'md'      |
| type     | string | no       | 'default' |
| onClick  | func   | no       | 'default' |
| children | node   | no       | -         |
